git笔记

  1. bare仓库是不会显示具体内容，都是以git内部方式存储

  2. error : git branch is currently checked out
    原因是git初始默认receive.denyCurrentBrach是拒绝接收push的，我们要向中心仓库push就需要设置它允许接收，在中心仓库输入：

  3. git reflog 可以查看所有的提交记录

  4. git log –pretty=oneline 只显示id和备注信息

  5. 用于回退版本，可以指定退回某一次提交的版本。

  6. 查看差异用diff

    • 缓存区 ↔ \leftrightarrow↔ 库(HEAD)：gitdiff –cached
    • 工作区 ↔ \leftrightarrow↔ 缓存区：gitdiff
    • 工作区 ↔ \leftrightarrow↔ 库(HEAD)：gitdiff HEAD – filename
    • 库 ↔ \leftrightarrow↔ 库：git diff 243550a 24bc01b

  7. GIT更新差异

    • git remote update 分支设置为跟踪远程分支，但不合并任何更改。
    • git fetch 将只更新您的分支’，但不会合并任何更改。
    • git pull 会更新和合并当前分支的任何远程更改。这将是您用来更新本地分行的人。
