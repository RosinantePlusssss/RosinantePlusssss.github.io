《马耳他之鹰》书评

发表于 2022-01-20 更新于 2022-01-24 分类于 读书
本文字数： 142 阅读时长 ≈ 1 分钟

最近又看回了侦探小说，已经忘记是在哪里看到的这本书加入的书架。看完感觉一般般，无论从古典侦探的角度来看，还是从社会派小说角度来看都是一般般。推理技巧并不能给读者带来愉悦，作案动机也不让感觉有其他的想法。唯一能提大概就是其算是硬汉派的开山鼻祖？在我看来也没有海明威笔下的渔夫来的硬汉。

《放学后》书评

发表于 2021-11-19 分类于 读书
本文字数： 188 阅读时长 ≈ 1 分钟

本身是一个弱推理的小说，虽说有一些小细节在暗示，但是都不能很好的进行推理。我觉得更多的是在讲述关于青少年犯罪的问题，他们的犯罪跟钱、权和欲望不大相关，更多的更像是一些情感类或者友情类的触动。如果从理智的方面来看，三的个老师并没有多严重的罪责，开头老师都无法确认到底有没有去偷窥，主角的事情离婚就能解决了。所以最后的反转在我看来是可有可无的，也就导致了这本书看起来会有点一般。
