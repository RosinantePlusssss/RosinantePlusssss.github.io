    在WEX5过程中的报错

    发表于 2019-09-19 更新于 2019-09-25 分类于 WEX5 本文字数： 167 阅读时长 ≈ 1 分钟

    一些遇到的错误，记录一下

    1.在APP打包后安装打开时出现了Appliction Error net::ERR_CONNECTION_REFUSED

    解决方案：在创建本地APP时勾选本地应用包含UI资源的选项。

    2.提示 Script error for: $model/UI2/system/components/justep/data/bizData http://requirejs.org/docs/errors.html#scripterror 类似的错误

    解决方案：打包APP时使用模式3,UISERVICE部署

    3.打包app时使用crosswalk引擎导致应用打不开；

    由于4.4及以上版本app默认使用的是手机内核的浏览器，如果集成crosswalk存在冲突，所以4.4及以上版本不能打包增强型浏览器的。
