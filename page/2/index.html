Git switch和Git checkout的区别

发表于 2022-09-19 更新于 2022-09-27 分类于 Git
本文字数： 198 阅读时长 ≈ 1 分钟

git checkout 和 git switch 都能对分支进行管理，但是 git checkout 同时还会对文件进行恢复。

所以在 git 的2.23版本中社区将 git checkout 命令进行拆分，新增了 git switch 和 git restore 用来替代 git checkout ，同时 git checkout 仍然保存用于提供更高级的选项来处理更复杂的情况。
